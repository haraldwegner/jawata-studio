# goja-studio v1.2.2

A correctness fix found while dogfooding v1.2.1.

## Fix — About box showed the wrong license

The in-app **About** dialog hardcoded "Released under the **MIT License**".
goja-studio is licensed under **AGPL-3.0** (see `LICENSE` and `NOTICE`; the
javalens-manager releases before v1.0.0 were MIT, but every GOJA release is AGPL).
The About box now reads **GNU AGPL-3.0** and links to the license text.

Also added the SPDX `license` field (`AGPL-3.0-only`) to `package.json` and
`Cargo.toml`, which were missing it — the omission is why the stale string went
unnoticed.

No functional changes.
