# jawata-studio v2.7.1 — polish

The finishing pass after the rename: the arch mark refined, documentation imagery
brought up to date, and a silent installer failure fixed.

## Changed

- **Dock icon refined:** rounded corners (as the goja-era icon had), the arch a bit
  lower and its line a bit fatter. The tray glyph is unchanged. All icon assets
  regenerated from the updated master.
- **Help screenshots refreshed** — all six now show the jawata-studio v2.7 UI
  (dashboard, memory, settings, tray).

## Fixed

- **`install.sh` failed silently when no published release was visible** (exactly the
  window in which v2.7.0 briefly sat as an unpublished draft): the no-match `grep`
  tripped `set -e` before the friendly error message could print. The script now
  reports what went wrong and where to look.

## Note

- The **automatic stop of surviving legacy goja residents** announced in the v2.7.0
  notes moves to a later release. When upgrading from goja-studio, the manual check
  still applies: quit the old app and `pgrep -af goja.jar` before first launch.
  Upgrades from jawata-studio v2.7.0 are unaffected.
