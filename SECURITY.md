# Security Policy

## Reporting a vulnerability

Please report security vulnerabilities **privately — do not open a public issue.**

Use GitHub's private vulnerability reporting: on the repository, go to the **Security** tab →
**Report a vulnerability**. This opens a private advisory visible only to the maintainers.

We aim to acknowledge reports within a few days and to keep you updated as we investigate and
remediate. Once a fix is available we'll coordinate disclosure with you and credit you (if you
wish) in the advisory.

## Supported versions

Security fixes target the **latest released version**. Older releases are addressed on a
best-effort basis.
